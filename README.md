# Off-Grid Shed

![Shed Exterior](05%20Wall%20Exterior/images/D03.svg)


## Features

* Holds 16' dimensional lumber.
* Goal: Plastic-free construction
* Goal: Compostable / Recyclable construction.
* Perfect for rainwater collection.
* Charred-wood siding.
* Off-grid solar power.
